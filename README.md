# Campus Maintenance & Infrastructure Triage Platform

A next-generation, high-performance web platform designed to streamline campus operational maintenance and infrastructure ticket routing. Built on a refined minimalist design philosophy, the platform leverages server-side intelligence, modern styling patterns, and intuitive interactive workflows to deliver a state-of-the-art triage dashboard for educational and corporate facilities.

## ✨ Key Features

The application is architected around two high-fidelity, user-centric dashboards designed to keep administrative workflows completely isolated and secure:

### 📢 Reporter Panel
*   **Intuitive Incident Filing:** Simple, structured reporting mechanism for physical failures, safety hazards, and resource requests.
*   **Base64 Evidence Uploads:** Submit visual confirmation or documentation attachments on the fly with real-time feedback.
*   **Interactive Floor Plotting:** Drag-and-drop locator marker system to specify exact campus zones, building levels, and rooms.

### 🛡️ Authority Panel
*   **Real-Time Category Management:** Browse reported categories efficiently with a responsive horizontal scrollbar/queue carousel that avoids text clipping.
*   **Integrated Diagnostics & Analytics:** High-level metrics tracking SLA compliance, open vs. closed ticket distribution, and triage urgency levels.
*   **Actionable Issue Board:** Swiftly change status states, assign maintenance personnel, and communicate updates.

### 🔑 Prototype-to-Production Auth
*   **Pre-filled Mock Bypasses:** Rapid stakeholder testing and developer previews with pre-populated credentials (`authority@example.com` / `password123`).
*   **Instant Social Bypasses:** Quick-access "Login with Google" authentication triggers.
*   **Audit-Ready Architecture:** Clean separation of concerns configured to easily swap client-side mock handlers with production authentication suites (e.g., JWT, Firebase, or Supabase).

## 🎨 Design Aesthetic: Minimalist Glassmorphism

The system sports a custom-tailored visual design optimized for modern displays:
*   **Frosted Glass Elements:** High-end translucent container overlays utilizing backdrop-filter properties (`backdrop-filter: blur()`).
*   **Vibrant Gradient Accents:** Smooth background color gradients showing through translucent sheets for subtle aesthetic depth.
*   **Modern Typography:** Elegant, high-contrast typography pairings incorporating spacious margins and robust negative space.
*   **Adaptive Theme Support:** Clean light and deep-space dark themes designed for visual comfort.
